Data sources and download approach - Core PCE inflation, CPI inflation, unemployment rate, NAIRU, unemployed level, and job openings come from FRED via the public fredgraph.csv endpoint. - Expected inflation (10-year CPI) comes from the Survey of Professional Forecasters (SPF) mean responses Excel file. - GSCPI comes from the NY Fed monthly data file (cached locally as CSV when available). - YouTube views are pulled from the Kworb monthly views table for the official “good 4 u” video. All downloads and the dataset assembly are handled by /Users/mtkonczal/Documents/good4u_inflation/scripts/build_phillips_data.py, which writes a single quarterly CSV at data/phillips_data.csv used below.

What this Quarto does - Downloads monthly CPI, unemployment, unemployed, and job openings from FRED; SPF 10-year expected CPI inflation from the Philadelphia Fed; and GSCPI from the NY Fed. - Aggregates all series to quarterly frequency. Core PCE inflation is computed as quarterly log differences annualized: (^{core}_t = 400(P^{core}t - P^{core}{t-1})). - Builds Phillips Curve regressions with expected inflation and two lags of realized inflation, then augments the curve with one variable at a time and with all variables together. - Uses the unemployment gap (u_t - u^*_t) (unemployment minus NAIRU).

Equations

Baseline Phillips Curve (quarterly):

[ ^{core}_t = _0 + _1 _t[^{10y}] + 2 ^{core}{t-1} + 3 ^{core}{t-2} + _4 (u_t - u^*_t) + _t ]

Augmented versions (one at a time):

[ ^{core}_t = _0 + _1 _t[^{10y}] + 2 ^{core}{t-1} + 3 ^{core}{t-2} + _4 (u_t - u^*_t) + Z_t + _t ]

where (Z_t) is one of: GSCPI, (v/u), ((1 + )) for Good 4 U, or (1{2021 t }).

All-augments model:

[ ^{core}_t = _0 + _1 _t[^{10y}] + 2 ^{core}{t-1} + 3 ^{core}{t-2} + _4 (u_t - u^*_t) + _1 _t + _2 (v/u)_t + _3 (1+_t) + 4 D{20212022} + _t ]

load_phillips_data <- function(path) {
  if (!file.exists(path)) {
    stop("Missing data/phillips_data.csv. Run scripts/build_phillips_data.py first.")
  }

  readr::read_csv(path, show_col_types = FALSE) |>
    mutate(qtr = as.yearqtr(as.Date(qtr_date)))
}

Model specifications

Code 
base_formula <- core_pce_pi ~ exp_pi10 + core_pce_pi_l1 + core_pce_pi_l2 + unrate_gap

fit_base <- lm(base_formula, data = phillips_df)
fit_gscpi <- lm(update(base_formula, . ~ . + gscpi), data = phillips_df)
fit_vu <- lm(update(base_formula, . ~ . + v_u), data = phillips_df)

have_youtube <- any(!is.na(phillips_df$good4u_views_log))
fit_youtube <- if (have_youtube) {
  lm(update(base_formula, . ~ . + good4u_views_log), data = phillips_df)
} else {
  NULL
}

fit_dummy <- lm(update(base_formula, . ~ . + y2021_2022), data = phillips_df)

fit_all <- if (have_youtube) {
  lm(update(base_formula, . ~ . + gscpi + v_u + good4u_views_log + y2021_2022), data = phillips_df)
} else {
  NULL
}

models <- list(
  "Baseline" = fit_base,
  "GSCPI" = fit_gscpi,
  "v/u" = fit_vu,
  "Good 4 U views (log1p)" = fit_youtube,
  "2021-2022 dummy" = fit_dummy,
  "All four" = fit_all
)

models <- models[!vapply(models, is.null, logical(1))]

coef_tbl <- imap_dfr(models, function(m, name) {
  broom::tidy(m) |>
    mutate(model = name)
}) |>
  filter(term != "(Intercept)") |>
  mutate(
    stars = case_when(
      p.value < 0.001 ~ "***",
      p.value < 0.01 ~ "**",
      p.value < 0.05 ~ "*",
      p.value < 0.1 ~ ".",
      TRUE ~ ""
    ),
    term = recode(
      term,
      exp_pi10 = "Expected inflation (10y)",
      core_pce_pi_l1 = "Core PCE inflation lag 1",
      core_pce_pi_l2 = "Core PCE inflation lag 2",
      unrate_gap = "Unemployment gap (u - u*)",
      gscpi = "GSCPI",
      v_u = "v/u",
      good4u_views_log = "Good 4 U views (log1p)",
      y2021_2022 = "2021-2022 dummy"
    ),
    estimate_fmt = sprintf("%.3f%s", estimate, stars),
    se_fmt = sprintf("(%.3f)", std.error)
  )

term_levels <- coef_tbl |>
  distinct(term) |>
  pull(term)

est_wide <- coef_tbl |>
  select(term, model, estimate_fmt) |>
  pivot_wider(names_from = model, values_from = estimate_fmt) |>
  mutate(term_order = match(term, term_levels), row_type = 1)

se_wide <- coef_tbl |>
  select(term, model, se_fmt) |>
  pivot_wider(names_from = model, values_from = se_fmt) |>
  mutate(term_order = match(term, term_levels), row_type = 2, term = "")

reg_table <- bind_rows(est_wide, se_wide) |>
  arrange(term_order, row_type) |>
  select(-term_order, -row_type) |>
  mutate(across(everything(), ~ ifelse(is.na(.), "", .)))
